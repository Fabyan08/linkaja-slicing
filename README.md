# 💻 LinkAja Website Slicing 

## 📌 Deskripsi Project

Project ini merupakan hasil slicing (recreate tampilan) dari website resmi **LinkAja (linkaja.id)** menggunakan HTML, CSS, dan JavaScript.
---

## 🎯 Halaman yang Dikerjakan

Project ini hanya mencakup beberapa bagian utama, yaitu:

* ✅ **Beranda (Homepage)**
* ✅ **Tentang Kami**
* ✅ **Fitur "Chat with an Expert" (UI saja)**

---
## 🙌 Catatan

Jika ada perbedaan kecil dari website asli (spacing, font, dll), itu merupakan bagian dari proses pembelajaran slicing.
